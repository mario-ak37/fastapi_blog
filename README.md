# fastapi-blog

A simple blog backend built with FastAPI.

## Overview
- REST-style API for blog content

## Requirements
- Python 3.14+
- `fastapi[standard]`

## Install
Using `uv`:
```bash
uv sync
```

Using `pip`:
```bash
python -m venv .venv
source .venv/bin/activate
pip install "fastapi[standard]"
```

## Run
```bash
uv run fastapi dev
```

Open `http://127.0.0.1:8000/` to verify the app is running.
